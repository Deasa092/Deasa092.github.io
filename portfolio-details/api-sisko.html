API Testing for Sistemtoko

Project information

  • Category: API Testing
  • Project date: January, 2024 - February, 2024
  • Tools: Postman
  • Project URL: https://github.com/Deasa092/sisko-api-testing

Testing the API for key features such as postage check, postage list, orders, product list, and product details, I ensured that the API responded to and managed various types of requests appropriately. I also verified consistency in the resulting responses, and I ensured that the list of shipping costs and products received from the API remained current and complete.
